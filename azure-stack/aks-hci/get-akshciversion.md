# Get-AksHciVersion

## Synopsis
Get the current Kubernetes version of Azure Kubernetes Service on Azure Stack HCI.

## Syntax

### Get current version
```powershell
Get-AksHciVersion  
                          
```

## Description
Get the current Kubernetes version of Azure Kubernetes Service on Azure Stack HCI.

## Examples

### Get current version
```powershell
PS C:\> Get-AksHciVersion
```
