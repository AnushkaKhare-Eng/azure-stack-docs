# Get-AksHciEventLog

## Synopsis
Gets all the event logs from the Azure Kubernetes Service on Azure Stack HCI PowerShell module.

## Syntax

```powershell
Get-AksHciEventLog
```

## Description
Gets all the event logs from the Azure Kubernetes Service on Azure Stack HCI PowerShell module.

## Examples

### Example
```powershell
PS C:\> Get-AksHciEventLog
```
