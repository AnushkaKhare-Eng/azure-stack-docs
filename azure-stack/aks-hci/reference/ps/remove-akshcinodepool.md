# Remove-AksHciNodePool

## Synopsis
Delete a node pool from a cluster.

## Syntax
```powershell
Remove-AksHciNodePool -clusterName <String>
                      -name <String>
```

## Description
Delete a node pool from a cluster.


## Example

```powershell
PS C:\> Remove-AksHciNodePool -clusterName mycluster -name nodepool1
```
