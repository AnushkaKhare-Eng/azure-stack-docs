# Get-AksHciUpdates

## Synopsis
List the available Kubernetes updates for Azure Kubernetes Service on Azure Stack HCI.

## Syntax

```powershell
Get-AksHciUpdates
```

## Description
List the available Kubernetes updates for Azure Kubernetes Service on Azure Stack HCI.

## Examples

### Example
```powershell
PS C:\> Get-AksHciUpdates
```
