# Update-AksHci

## Synopsis
Update the Azure Kubernetes Service host to the latest Kubernetes version.

## Syntax

```powershell
Update-AksHci
```

## Description
Update the Azure Kubernetes Service host to the latest Kubernetes version.

## Examples

### Example
```powershell
PS C:\> Update-AksHci
```
