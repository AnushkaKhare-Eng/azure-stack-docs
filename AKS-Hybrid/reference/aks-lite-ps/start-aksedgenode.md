# Start-AksEdgeLinuxNode

## Synopsis

Starts the Linux node VM if it's currently stopped.

## Syntax

```
Start-AksEdgeLinuxNode
```

## Description

The Start-AksEdgeLinuxNode cmdlet starts the virtual machine.
No action is taken if the virtual machine is already started.

## Examples

```
Start-AksEdgeLinuxNode
```
