# Stop-AksIotLinuxNode

## Synopsis
Stops the Linux node VM if it's currently started.

## Syntax

```
Stop-AksIotLinuxNode
```

## Description
The Stop-AksIotLinuxNode cmdlet stops the virtual machine.
No action is taken if the virtual machine is already stopped

## Examples


```
Stop-AksIotLinuxNode
```
