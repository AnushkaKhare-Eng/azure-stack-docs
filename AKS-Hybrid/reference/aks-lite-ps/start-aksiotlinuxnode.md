# Start-AksIotLinuxNode

## Synopsis

Starts the Linux node VM if it's currently stopped.

## Syntax

```
Start-AksIotLinuxNode
```

## Description

The Start-AksIotLinuxNode cmdlet starts the virtual machine.
No action is taken if the virtual machine is already started.

## Examples

```
Start-AksIotLinuxNode
```
