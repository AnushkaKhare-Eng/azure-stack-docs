# Get-AksIotLinuxNodeName

## SYNOPSIS
Gets the Linux VM's hostname

## SYNTAX

```
Get-AksIotLinuxNodeName [<CommonParameters>]
```

## DESCRIPTION
The Get-AksIotLinuxNodeName queries the Linux virtual machine's current hostname from wssdagent.

## EXAMPLES

### EXAMPLE 1
```
Get-AksIotLinuxNodeName
```
