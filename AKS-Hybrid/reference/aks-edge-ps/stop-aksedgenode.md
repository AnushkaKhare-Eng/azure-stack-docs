# Stop-AksEdgeNode

## Synopsis
Stops the node VM if it's currently started.

## Syntax

```powershell
Stop-AksEdgeNode
```

## Description
The Stop-AksEdgeNode cmdlet stops the virtual machine.
No action is taken if the virtual machine is already stopped

## Examples


```powershell
Stop-AksEdgeNode
```
