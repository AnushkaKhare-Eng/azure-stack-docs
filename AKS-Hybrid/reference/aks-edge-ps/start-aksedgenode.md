# Start-AksEdgeNode

## Synopsis

Starts the node VM if it's currently stopped.

## Syntax

```powershell
Start-AksEdgeNode
```

## Description

The Start-AksEdgeNode cmdlet starts the virtual machine.
No action is taken if the virtual machine is already started.

## Examples

```powershell
Start-AksEdgeNode
```
