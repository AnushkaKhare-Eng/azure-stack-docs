# Get-AksEdgeDeploymentInfo

## Synopsis
Gets AksEdge deployment information.

## Syntax

```powershell
Get-AksEdgeDeploymentInfo
```

## Description
Gets AksEdge deployment information.
The function returns networking and node information of the local
AksEdge deployment.
